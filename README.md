# Watchuru
Watchuru is a platform that provide variety of movies. Movies which worth to watch as they are produced from valuable books and narratives.  


Technologies used:

1-Java <br/>
2-Microservices <br/>
3-Spring boot <br/>
4-Spring Cloud<br/>
5-Hibernate<br/>
6-H2 DB<br/>
